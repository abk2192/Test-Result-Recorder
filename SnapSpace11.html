Keyboard Shortcuts

        
    
          
  • Ctrl + Shift + F : Search all projects
    • 
          
  • Ctrl + S : Export project
    • 
          
  • Ctrl + O : Import project
    • 
          
  • Alt + N : Add new item
    • 
          
  • Alt + T : Add new tab
    • 
          
  • Alt + R : Rename tab
    • 
          
  • Alt + F : Add new field to current item
    • 
        

        
Tips

        
    
          
  • Drag & Drop: Reorder projects, tabs, and items by dragging them.
    • 
          
  • Images: Paste screenshots directly into the Notes area (Ctrl+V). Double-click to zoom.
    • 
          
  • Cross-Project Move: Click the Move icon on any item to send it to another project.
