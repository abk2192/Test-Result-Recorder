menu_book How to Use the Test Recorder

      

        
    
          
  • Header Editing: Click directly on a scenario's name in the header to edit it instantly.
    • 
          
  • Drag and Drop: Drag tabs left and right to reorder them. Drag scenarios up and down to reorder them within a tab!
    • 
          
  • Templates: Save commonly used scenarios as Templates. Apply them to quickly start new tests. Share them via JSON import/export.
    • 
          
  • File Previews & Copying: Clicking an attached file opens a reading window where you can copy the text. Clicking the copy icon next to any attachment copies the entire file block.
    • 
          
  • Compare Mode: Click "Compare" to open a split-pane view, perfect for evaluating "Before" and "After" evidence side-by-side with synchronized scrolling.
    • 
          
  • Selective Export: Click "Export" to pick exactly which tabs and scenarios you want to share with your team.
    •
